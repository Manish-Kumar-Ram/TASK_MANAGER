# 🚀 Task Manager App

A modern and interactive **Task Manager Application** built using **Vanilla JavaScript, HTML, and CSS**. This project demonstrates core frontend development concepts including **CRUD operations, Event Delegation, Local Storage, Live Search, Theme Switching, and Dynamic DOM Manipulation**.

---

## 📌 Overview

The Task Manager App helps users organize and manage their daily tasks efficiently. It provides a clean user interface with persistent data storage, real-time search functionality, and customizable themes.

---

## ✨ Features

### 📝 Task Management (CRUD)

* ➕ Add new tasks
* 👀 View all tasks
* ✏️ Edit existing tasks
* 🗑️ Delete tasks

### 🔍 Live Search

* Real-time task filtering
* Instant search results while typing

### 🌙 Dark / ☀️ Light Mode

* One-click theme switching
* User preference saved in Local Storage

### ☑️ Task Completion

* Mark tasks as completed using checkboxes
* Improved task tracking experience

### 💾 Local Storage Integration

* Automatically saves tasks in the browser
* Data persists after page refresh or browser restart

### ⚡ Dynamic UI Rendering

* Tasks are generated dynamically using JavaScript
* No page reloads required

### 🎯 Event Delegation

* Efficient event handling for dynamically created elements
* Improved performance and scalability
